zou-jenkins-tools
=================

## 説明！

[zou-android](https://github.com/yamacraft/zou-android)のプロジェクトのJenkins関連で使うファイル群です。

### 詳細！

- magicpod/
    - TRIDENT社が開発中のツール「Magic Pod」の設定ファイルがあります
- shell/
    - jenkinsで利用するシェルです
        - run-build.sh
            - zouのビルド＋デプロイ
        - run-magicpod.sh
            - Magic Podの実行
